Page creation process and background

Start

The page started out with basic HTML. I added in blank links to information pages of many different file extensions. File extensions really interest me, I used to make lists of thousands of them. Now I get to put this work to use and enjot it.

The page hasn't changed from this point yet

Images

I drew the outlines of the images with MS-PAINT. Most of the imaegs needed some touchup and finalizations with GIMP (GNU Image Manipulation Program) so I filled in textures, and added in text. There were only a few images that didn't need modification.

I am not the best graphical artist, so these will just be the classical icons. I will get help creating better icons in the future. The original icons will still be in archives, and a separate option for page viewing.

Purpose

The purpose of this page is to keep reference of file extensions, what they do, how they work, and info on them. It will take a long time to do this, as there are millions of file extensions, and the list keeps on going. People will need reference of them from time to time.

The system will mostly be Wiki-based, and the Wiki-engine will be a separate distribution from the SNU wiki engine.
